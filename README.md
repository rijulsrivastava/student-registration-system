<h1> student-registration-system </h1>
<p>Student registration portal that store user entered student data which can be modified or removed later and data is stored in local storage to ensure data retention even after page reloading</p>



<h2> Objective </h2>
<ol>
  <li>To develop a responsive student registration portal</li>
  <li>To store student details including Student name, student ID, email ID, and Contact number</li>
  <li>Able to edit record</li>
  <li>Able to delete particular record</li>
  <li>Data should retain after reloading the page</li>
</ol>
